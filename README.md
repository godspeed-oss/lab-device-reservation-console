\# 实验室设备预约系统（控制台版）



这是一个使用 Java 编写的实验室设备预约管理系统控制台版本，用于练习 Java 面向对象、集合、业务逻辑判断和 Git/GitHub 版本管理。



\## 项目功能



当前已实现：



\- 查看实验室设备列表

\- 查看全部预约记录

\- 新增预约记录

\- 查询指定设备的预约记录

\- 判断设备是否存在

\- 判断设备是否处于可预约状态

\- 判断同一设备在同一天的预约时间段是否冲突

\- 校验预约时间段格式是否合法



\## 技术点



本项目练习了以下 Java 基础能力：



\- 类与对象

\- 构造方法

\- private 属性封装

\- getter 方法

\- ArrayList 集合

\- Scanner 控制台输入

\- while 循环

\- switch 分支

\- 方法拆分

\- 字符串处理

\- 时间段冲突判断逻辑

\- Git 版本管理

\- GitHub 远程仓库同步



\## 核心类说明



\### Device



表示实验室设备。



主要字段：



\- id：设备编号

\- name：设备名称

\- type：设备类型

\- status：设备状态



\### Reservation



表示预约记录。



主要字段：



\- id：预约编号

\- deviceId：设备编号

\- userName：预约人

\- date：预约日期

\- timeSlot：预约时间段
